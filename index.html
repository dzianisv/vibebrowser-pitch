Architecture / delivery

Extension (MVP)

Chromium extension. DOM injection + browser control. Lowest friction for users.

Vibe Browser

Custom browser build (based on Google's Chromium open source project) for deeper context access and background capabilities beyond extension APIs. Hard to convience users to use another browser

Cloud agent

Headless Chromium (Playwright) + LLM orchestration for long-running, reliable automations.
